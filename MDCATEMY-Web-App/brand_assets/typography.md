# MDCATEMY Typography System

## Core Philosophy

Typography should feel:
- Bold
- Confident
- High-impact
- Slightly aggressive (warrior tone)

Avoid soft, friendly, or overly corporate typography.

---

## Base System (Recommended)

Primary Font Pair:
- Headings: Poppins (600–900)
- Body: Inter (400–500)

Monospace:
- JetBrains Mono (for MCQs, numbers, technical UI)

---

## Heading Scale

- H1: 36–48px (Hero / major sections)
- H2: 28–34px
- H3: 22–28px
- H4: 18–24px

---

## Body Text

- Default: 16px
- Large: 18px
- Small: 14px

Line-height should be comfortable but tight enough to feel sharp.

---

## Labels & Buttons

- Buttons: uppercase, slight letter spacing (0.04em)
- Labels: small caps / uppercase optional
- UI text should feel structured, not decorative

---

## IMPORTANT: Flexibility for AI

Claude is ALLOWED to:
- Experiment with font combinations (within premium feel)
- Adjust sizes slightly based on layout
- Use strong typographic contrast (big headlines, tight spacing)
- Introduce creative hierarchy (editorial style)

Claude MUST:
- Maintain bold, high-contrast hierarchy
- Keep readability strong
- Avoid generic UI typography (no default boring layouts)

Claude MUST NOT:
- Use overly playful or childish fonts
- Use soft/pastel typography styles
- Flatten hierarchy (everything same size)

---

## Style Direction

Typography should feel like:
- A serious product
- A focused system
- A disciplined environment

NOT:
- A blog
- A startup template
- A generic dashboard
