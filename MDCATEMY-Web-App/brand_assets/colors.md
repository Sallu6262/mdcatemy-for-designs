# MDCATEMY Color System

## Core Theme
- Dark theme ONLY. No light mode anywhere.
- High contrast: Yellow on Black is primary identity.
- UI must feel bold, sharp, and premium.

---

## Primary Colors

- Primary Accent: #FFC600 (MDCATEMY Yellow)
- Hover Accent: #E6A800 (Deep Gold)

---

## Background System

- Primary Background: #181A18 (Warrior Black)
- Secondary Surface: #222422 (Dark Charcoal)

RULE:
- No white backgrounds
- No light UI sections
- All layouts must stay dark

---

## Text Colors

- Primary Text: #FFFFFF
- Secondary Text: #CCCCCC
- Subtle/Muted: #F0F0F0 (use very sparingly)

---

## Functional Colors

- Success: #28A745
- Error: #D9534F
- Warning: #FFA500
- Info: #0070C0

---

## Usage Principles

- Yellow = action, highlights, energy (CTA, active states)
- Black/Charcoal = structure and depth
- Avoid overusing yellow → use it for impact
- Maintain strong contrast at all times

---

## Creative Flexibility

Claude MAY:
- Use gradients involving yellow and dark tones
- Add glow, shadow, or layered effects

Claude MUST NOT:
- Introduce random new color palettes
- Break dark theme consistency
- Use soft pastel or generic SaaS color schemes
